☁ 클라우드 동기화 설정

현재 상태: 미설정
집·회사 PC 간 데이터를 동기화합니다. 무료 Supabase 프로젝트로 한 번만 설정하세요.
  1. supabase.com 가입 → New Project 생성(무료)
  2. 좌측 SQL Editor에서 아래 SQL 실행(테이블+권한 생성)
  3. Project Settings → API에서 Project URL과 anon public 키 복사
  4. 아래 두 칸에 붙여넣고 저장 → 두 PC 모두 같은 값 입력
create table if not exists schedule_sync(
  id text primary key,
  data jsonb,
  updated_at bigint
);
alter table schedule_sync enable row level security;
create policy "anon_all" on schedule_sync
  for all to anon using (true) with check (true);
연결 해제해도 이 PC의 로컬 데이터는 보존됩니다.

  3. OAuth 동의 화면 → 외부 → 본인 이메일을 테스트 사용자로 추가
  4. 사용자 인증 정보 → OAuth 클라이언트 ID → 웹 애플리케이션 생성
  5. 승인된 자바스크립트 원본에 이 페이지 주소 입력 (예: )
  6. 발급된 클라이언트 ID를 아래에 붙여넣고 연결
